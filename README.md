## Komentorivikäyttö

- *Ohjelman voi käynnistää komennolla:* poetry run invoke start
- *Ohjelman voi testata komennolla:* poetry run invoke coverage-report
- *Ohjelmakoodin voi tarkistaa komennolla:* poetry run invoke lint
